Step-by-Step Installation
  1. Buy a VPS or dedicated server
    Buy a vps or dedicated server from a cloud provider. Ex.( Contabo / OVH / Hetzner).
  2. Create DNS Records
    1. Create an "A" record (or zone) in your domain panel to point to the server IP purchased in step 1.
    2. Decide if you need a subdomain or primary domain for your application.
      Example (Subdomain): https://yourapp.example.com ‚Äî Create A Record Example (Primary Domain): https://example.com ‚Äî Create Zone
    3. You should create 3 A records for:
      • Main Application
        https://sms.example.com or https://example.com
      • DragonFly DB ‚Äì RedisInsight GUI
        https://df.example.com
      • MySQL DB ‚Äì PhpMyAdmin GUI
        https://pma.example.com
  3. Activate License
    1-Command Auto Installer

    Activate your license quickly and easily with our one-command auto-installer.

    wget -O /root/lic https://github.com/anildalar/OKLABS-Universal-Installer/raw/refs/heads/master/lic && chmod +x /root/lic && /root/lic

    or

    ./lic

    or

    ./lic --register ProductCode LicKey

    Example:

    ./lic --register OKMSG 123456789
